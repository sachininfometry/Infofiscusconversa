# Infometry Product Templates

Separate WordPress plugin for Infometry product page templates.

The homepage plugin is intentionally separate. Product-specific design, assets, and template logic live here so future product page work does not disturb the completed homepage plugin.

## Templates

- `INFOFISCUS Conversa Product` (`templates/page-infofiscus-conversa.php`)

## Local Preview

Open `preview-conversa.html` through a local static server to preview the design with the bundled assets.

## WordPress Usage

1. Upload and activate this plugin.
2. Edit the target WordPress page.
3. In **Page Attributes > Template**, select **INFOFISCUS Conversa Product**.
4. Click **Update**.
5. Clear WordPress/plugin cache if the old page still appears.
